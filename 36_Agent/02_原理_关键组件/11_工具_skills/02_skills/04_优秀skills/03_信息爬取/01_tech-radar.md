tech-radar——帮我每天找选题的那双眼睛写自媒体最大的问题不是"怎么写"，是"今天写什么"。tech-radar 是我搓的一个情报雷达——五个信息源并行采集：GitHub Trending、Product Hunt、Hacker News、张咋啦 AI、Follow Builders。每天自动跑一次，AI 分析师独立研判每一条线索，最后由一个"主编 Agent"跨源综合，给我一份日报。我不用再每天刷 X、刷 HN、刷公众号——我只看这一份日报，然后从里面挑选题。它能做什么：·每天早上自动生成一份 AI 行业日报（5 源综合）·识别出"可能值得写"的选题（按转发率、技术含量、故事性打分）·把同一事件在不同源的声音做对比（比如一个工具在 GitHub 上被吹，在 HN 上被喷）开源地址：https://github.com/OrangeViolin/tech-radar
