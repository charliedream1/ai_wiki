ip-coach——写这篇复盘时顺手做的一个 skill写这篇的过程中我有一个担心——文章你读完可能会觉得"有道理"，但合上之后，大概率还是不会真的动手。我不想让这篇只是一个"看过了就算"的文章。所以我把这篇文章背后的方法论（第四节的四步法）做成了一个 skill，叫 ip-coach。它是一个陪伴型访谈工具——装上之后，你对你的 agent 说一句"我想起号"，它会用 7 个阶段的提问陪你走完一次完整的自我定位：

访谈完成后会生成一份《起号档案》保存到你的项目里，然后自动把你的第一个里程碑交棒给 /goal skill——从此你的 agent 会在日常对话里参考这份档案，也会帮你把日常任务安排起来。这个 skill 的铁律只有一条：不替你做决定，只追问到你自己没办法再对自己含糊为止。开源地址：https://github.com/OrangeViolin/ip-coach

安装后在 Claude Code 里说"我想起号"或 /起号 就会触发。
