scansci-pdf 不是去一个地方找 PDF，它是同时去 13 个地方找。Sci-Hub、Unpaywall、OpenAlex、SemanticScholar、Crossref、DOAJ、EuropePMC、CORE、PMC、LibGen、出版商直链、OA 发现网络，13 个源同时开跑，谁先返回有效 PDF 就用谁的。

如果你非要手动操作，那就是 pip install -U scansci-pdf，然后在配置文件里加上 MCP 服务器地址。但说实话，没必要。一句话能搞定的事，为什么要记命令？当然了，codex/claude code/opencode/z code都可以一句话安装。

有些论文学校有订阅，但必须通过校园网或者 WebVPN 才能访问。scansci-pdf 支持 100 多个中国高校的 WebVPN，包括清华、北大、浙大、上交、复旦、南大、中科大等。
