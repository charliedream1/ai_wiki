# 1. PydanticAI

pydantic_ai 包、核心依赖项以及使用 PydanticAI 中包含的所有模型所需的库。如果您想使用某个特定模型，可以安装 "slim "版本的 PydanticAI。

PydanticAI 的 8 个重要组成部分
-  Agents
-  Models （模型）
-  Dependencies （依赖）
-  Function Tools  -  函数工具
-  Results  -  结果
-  信息和聊天记录
-  测试和评估
-  调试和监控
