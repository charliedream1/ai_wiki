AI技能和代理让每个工程单元的工作比上一个更轻松。

理念
每个工程单元的工作都应该让后续单元变得更容易——而不是更难。

传统开发会积累技术债务。每一个功能都会增加复杂性。每次修复错误都会留下更多本地知识，之后需要有人重新发现。代码库变得更大，上下文变得更难掌握，下一次变更也变得更慢。

化合物工程则颠倒了这一点。80%的项目在规划和审查，20%在执行中：

在编写代码前要彻底规划，并且/ce-brainstorm/ce-plan
复习以发现问题并校准判断，并结合/ce-code-review/ce-doc-review
将知识编码以便可重复使用，/ce-compound
保持高质量，这样以后改动就更容易了
重点不是礼节。关键是筹码。一个好的头脑风暴让计划更清晰。一个好的计划让执行更简单。一个好的评审能抓住模式，而不仅仅是漏洞。一个好的复合音意味着下一个经纪人不必从零学到同样的教训。
