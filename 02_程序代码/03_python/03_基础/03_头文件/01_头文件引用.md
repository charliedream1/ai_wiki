如果引用自己创建的python文件夹下的文件中的方法，无法找到该文件，需要用如下方法引入根目录，程序就能找到对应的文件夹了

```python
import os
import sys

path = os.getcwd()
sys.path.append(os.path.abspath(path + ('/..' * 3)))
```

如果引入回退2级的目录

```python
import os
import sys

root_path = os.path.join(os.path.dirname(__file__), '..', '..')
sys.path.append(root_path)
```
